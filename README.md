# 💸 Expense Tracker (Spring Boot)

A simple web-based Expense Tracker built using **Spring Boot**, **Thymeleaf**, and **PostgreSQL**.
It allows users to manage daily expenses with basic CRUD operations.


## 🚀 Features

* ➕ Add new expenses
* 📋 View all expenses
* ✏️ Edit existing expenses
* ❌ Delete expenses
* 💾 Data stored in PostgreSQL database



## 🛠️ Tech Stack

* **Backend:** Spring Boot
* **Frontend:** Thymeleaf, HTML, CSS
* **Database:** PostgreSQL
* **Build Tool:** Maven



## 📂 Project Structure

src/
 └── main/
     ├── java/com/example/expensetracker/
     │   ├── controller/      # Handles HTTP requests
     │   ├── model/           # Entity classes
     │   ├── repository/      # Database operations (JPA)
     │   └── ExpenseTrackerApplication.java
     └── resources/
         ├── templates/       # HTML files (Thymeleaf)
         └── application.properties
